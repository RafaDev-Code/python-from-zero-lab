# Pieza 08 - Bucles

## Objetivo

Repetir tareas sin escribir la misma linea muchas veces.

## Concepto

`for` recorre una coleccion.

`while` repite mientras una condicion sea verdadera.

## Formula

```python
for elemento in lista:
    print(elemento)
```

```python
while condicion:
    accion
```

## Ejemplo

```python
nombres = ["Ana", "Luis", "Marta"]

for nombre in nombres:
    print(nombre)
```

## Tu problema

En `2-main.py`, crea una lista de productos.

Con `for`, mostra el nombre y el stock de cada producto.

Despues arma un `while` simple que muestre un menu hasta que el usuario escriba
`salir`.
