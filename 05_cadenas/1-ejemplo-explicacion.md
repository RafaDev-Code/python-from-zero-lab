# Pieza 05 - Cadenas

## Objetivo

Trabajar textos con metodos de strings.

## Concepto

Un string es texto.

Los metodos son acciones que se aplican sobre un dato.

## Formula

```python
texto.metodo()
```

## Ejemplo

```python
nombre = "  ana perez  "

print(nombre.strip())
print(nombre.upper())
print(nombre.title())
```
