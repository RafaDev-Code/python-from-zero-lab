# Pieza 17 - Proyecto final gestor de stock y ventas

## Objetivo

Unir las piezas anteriores en una app de consola.

## Concepto

Un proyecto final no es un tema nuevo.

Es integracion:

- datos
- listas y diccionarios
- inputs
- condicionales
- bucles
- funciones
- archivos
- errores

## Formula mental

```text
menu -> opcion -> accion -> actualizar datos -> mostrar resultado -> repetir
```

## Ejemplo de menu

```python
print("1 - Listar")
print("2 - Agregar")
print("3 - Salir")
```

Ese ejemplo solo muestra la idea de menu. No resuelve el gestor.

## Tu problema

En `2-main.py`, arma una primera version del gestor.

Debe permitir:

- cargar productos
- listar productos
- vender un producto
- descontar stock
- salir

Primero puede guardar todo en memoria.

Despues, si esta claro, agregamos CSV a mano.
