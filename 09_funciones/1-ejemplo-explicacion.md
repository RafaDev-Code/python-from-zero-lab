# Pieza 09 - Funciones

## Objetivo

Separar acciones con nombres claros.

## Concepto

Una funcion guarda un bloque de codigo para reutilizarlo.

Puede recibir datos y puede devolver un resultado.

## Formula

```python
def nombre_funcion(parametro):
    return resultado
```

## Ejemplo

```python
def calcular_doble(numero):
    return numero * 2

resultado = calcular_doble(5)
print(resultado)
```
