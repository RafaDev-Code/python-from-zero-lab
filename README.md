# python-from-zero-lab

Repo para aprender Python con piezas chicas.

Cada pieza tiene solo:

```text
1-ejemplo-explicacion.md
2-main.py
```

`1-ejemplo-explicacion.md` es el pizarron: concepto, explicacion y ejemplo.

`2-main.py` queda vacio: ahi resolvés vos a mano.

## Metodo

1. Lees el pizarron.
2. Escribis tu solucion en `2-main.py`.
3. Ejecutas con `python3 2-main.py`.

## Regla

El ejemplo muestra la formula. No muestra tu respuesta.
