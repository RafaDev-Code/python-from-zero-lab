# Pieza 03 - Datos compuestos

## Objetivo

Agrupar varios datos en una sola estructura.

## Concepto

Una lista guarda varios valores en orden.

Un diccionario guarda datos con clave y valor.

## Formula

```python
lista = ["valor1", "valor2"]
diccionario = {"clave": "valor"}
```

## Ejemplo

```python
colores = ["rojo", "azul", "verde"]

persona = {
    "nombre": "Ana",
    "edad": 32
}

print(colores)
print(persona["nombre"])
```
