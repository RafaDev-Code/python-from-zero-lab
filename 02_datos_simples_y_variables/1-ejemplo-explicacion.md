# Pieza 02 - Datos simples y variables

## Objetivo

Guardar datos simples con nombres claros.

## Concepto

Una variable es un nombre que apunta a un valor.

Python tiene datos simples como texto, numeros enteros, numeros decimales y
booleanos.

## Formula

```python
nombre_variable = valor
```

## Ejemplo

```python
cliente = "Ana"
edad = 32
saldo = 1500.50
activo = True

print(cliente)
print(edad)
print(saldo)
print(activo)
```
