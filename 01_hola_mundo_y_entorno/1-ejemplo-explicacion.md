# Pieza 01 - Hola mundo y entorno

## Objetivo

Mostrar textos en la terminal usando Python.

## Concepto

`print()` muestra un dato por pantalla.

Python ejecuta el archivo de arriba hacia abajo.

## Formula

```python
print("texto que quiero mostrar")
```

## Ejemplo

```python
print("Bienvenido al sistema")
print("Cargando datos")
```

Salida:

```text
Bienvenido al sistema
Cargando datos
```
